# 方法一

> 
思路：对于这种，“**最早**”类型的题，指针很好用。（时间复杂度O（n）--&gt;实际是n，也就是一遍遍历）
双指针法则：一个指当前只出现一次的位置（j），一个指当前位置（i）。
一个长度为26的临时数组，存字符出现的次数。一旦超过1就说明有重复。那j就继续跳，直到跳到不重复为止。
